# Obsidian CLI Skill for Coding Agents

Skill reusable para agentes como Codex y Claude Code, enfocada en automatizar workflows de Obsidian desde terminal.

## Contenido

- `obsidian-cli/SKILL.md`: instrucciones de uso y flujo operativo
- `obsidian-cli/references/commands.md`: referencia de comandos CLI
- `obsidian-cli/agents/openai.yaml`: metadata de interfaz para agentes compatibles

## Uso en Codex

Copiar o enlazar `obsidian-cli/` dentro de tu carpeta de skills (`$CODEX_HOME/skills`).

## Uso en otros agentes

Agentes que soportan skills por carpeta pueden apuntar directamente a `obsidian-cli/`.
El archivo principal es `SKILL.md` con frontmatter y guia operativa.
